# Notas:

## **Usage**: 

### *npm install*
```

Opciones:
    --help     Muestra ayuda                    [booleano]
    --version  Muestra número de versión        [booleano]  
-b, --base     Base of multiplication table     [número] [requerido]  
-l, --list     Print table on the console       [booleano] [defecto: false]  -m, --minimum  Table start with this number     [número] [defecto: 1]  -x, --maximum  Table end with this number           [número] [defecto: 10]

```
